# AtomQuest Video Support Platform

## Overview

AtomQuest Video Support Platform enables support agents and customers to communicate through a browser-based support session.

## Features

- Session Creation
- Join via Invite Link
- Real-Time Chat
- Webcam Access
- Participant Count
- End Session
- Session History API

## Tech Stack

Frontend:
- React
- React Router
- Axios

Backend:
- Node.js
- Express.js
- Socket.IO

## How To Run

Backend:

npm install
node server.js

Frontend:

npm install
npm run dev
